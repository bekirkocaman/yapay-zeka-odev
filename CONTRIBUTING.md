# Katkıda Bulunma

Katkılarınız için teşekkürler.

## Nasıl katkı verilir?

1. Bu repoyu fork edin.
2. Yeni bir branch oluşturun: `git checkout -b feature/aciklama`
3. Değişikliklerinizi commit edin: `git commit -m "feat: kisa aciklama"`
4. Branch’i push edin: `git push origin feature/aciklama`
5. Pull Request açın.

## Kurallar

- `.env`, `credentials.json`, `token.json` veya gerçek API anahtarlarını **commit etmeyin**.
- Büyük dosyalar (PDF, `chrome_profile`, `venv`) eklemeyin.
- Mümkünse mevcut kod stiline uygun, odaklı değişiklikler yapın.

## Hata bildirimi

[Issue açarken](https://github.com/bekirkocaman/yapay-zeka-odev/issues/new/choose) mümkünse hata mesajı, işletim sistemi ve adımları ekleyin. Ekran görüntüsü paylaşabilirsiniz; **şifre veya API anahtarı** paylaşmayın.
