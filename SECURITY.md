# Güvenlik Politikası

## Desteklenen sürümler

| Sürüm | Destek |
|-------|--------|
| `main` branch | Aktif |

## Gizli bilgiler

Aşağıdakileri **asla** issue, PR veya commit içinde paylaşmayın:

- `.env` içeriği
- `credentials.json` / `token.json`
- Anthropic, Google veya Gemini API anahtarları
- Okul hesabı şifreleri

## Güvenlik açığı bildirimi

Bir güvenlik sorunu bulduysanız lütfen **public issue açmayın**. Bunun yerine GitHub üzerinden [Security Advisory](https://github.com/bekirkocaman/yapay-zeka-odev/security/advisories/new) oluşturun veya repoda issue açmadan doğrudan iletişime geçin.

Anahtarınız yanlışlıkla public repoya yüklendiyse hemen:

1. Anahtarı ilgili panelden **iptal edin / yenileyin**
2. Commit geçmişinden kaldırmak için [GitHub secret scanning](https://docs.github.com/en/code-security/secret-scanning) yönergelerini izleyin
